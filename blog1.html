Selectors

CSS Selectors selects HTML elements to which we want to apply styles.

They can be divided into five categories.They are:

  • Simple Selectors
  • Combinational Selectors
  • Pseudo-class Selectors
  • Pseudo-element Selectors
  • Attribute Selectors

Simple Selectors:

Selects elements based on name, id, class.

  • Universal Selector

    Selects all elements in the entire document or can be restricted to specific namespace.

    Syntax: *

    Example: * will select all the elements in the document.

  • Type Selector

    Selects all elements that has given node name.

    Syntax: elementname

    Example: button will select < button> element.

  • Class Selector

    Selects all elements that has given class attribute.

    Syntax: .classname

    Example: .list will select the element that has a class "list" in the document.

  • ID Selector

    Selects all elements that has given ID attribute.ID should be unique

    Syntax: #idname

    Example: #main will select the element that has ID "main" in the document.

Combinational Selectors:

A CSS Selector that has more than one simple selector.

  • Descendant Combinator

    The " "(space) combinator selects nodes that are descendents of the first element.

    Syntax: A B

    Example: div p will match all < p> elements that are inside a < div> element.

  • Child Combinator

    The > combinator selects nodes that are direct children of the first element.

    Syntax: A > B

    Example: div > p will match all < p> elements that are directly inside a < div> element.

  • Genral Sibling Combinator

    The ~ combinator selects siblings. i.e., second element follows the first one.

    Syntax: A ~ B

    Example: div ~ p will match all < p> elements that follow < div> immediately or not.

  • Adjacent Sibling Combinator

    The + combinator selects adjacent siblings. i.e., second element directly follows the first.

    Syntax: A B

    Example: div + p will match all < p> elements that directly follow < div> element.

  • Column Combinator

    The || combinator selects nodes that belong to a column.

    Syntax: A || B

    Example: col || td will match all < td> elements that belong to scope of < col>.

    • Pseudo-class Selectors

    Used to define special state of an element

    Example use cases:

    • To style differently when mouse hovers over an element

      Example: a:hover { color: red; }

    • To style when an element gets focused

      Example: a:active{ color: green; }

    • To style visited and unvisited links differently

      Example: a:link { color: #FF0000; }, a:visited{ color: #FF00F0; }

    Pseudo-element Selectors:

    Used to style specific parts of an element.

    Example use cases:

  • To style first letter, line of an element.

    Example: p::first-line { -- }

  • To insert contenr before, after the content of an element.

    Example: h1::before { content: -- }

    • Attribute Selectors:

  • Example use cases:

    Styles HTML elements that have specific attributes or attribute values.

    Syntax: [attr], [attr=value], [attr~=value], etc.

    Example: [autoplay] will select the element whose attribute was set to "autoplay" in the document.
